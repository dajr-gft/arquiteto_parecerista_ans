## 1. PROPOSTA TÉCNICA

### 1.1 Entendimento da Solução

**1.1.1 Descreva seu entendimento sobre os objetivos do projeto:**

Compreendemos que o projeto visa implementar uma solução de análise automatizada de propostas utilizando IA generativa (Google Gemini/Vertex AI) para otimizar o processo de avaliação de fornecedores. A solução deve processar documentos técnicos e comerciais, identificar não conformidades, gerar pareceres estruturados e integrar-se com sistemas corporativos existentes.

**1.1.2 Qual a arquitetura proposta?**

- **Frontend**: Aplicação web em Streamlit para interface de usuário
- **Backend**: FastAPI para APIs REST
- **IA/ML**: Google Vertex AI com modelo Gemini 1.5 Pro
- **Orquestração**: Google Cloud Run para containers
- **Armazenamento**: Google Cloud Storage para documentos
- **Banco de Dados**: Cloud SQL (PostgreSQL) para metadados
- **Segurança**: Identity Platform + Cloud IAM
- **Observabilidade**: Cloud Logging + Cloud Monitoring

**1.1.3 Quais tecnologias serão utilizadas?**

- Python 3.10+
- FastAPI 0.104+
- Streamlit 1.28+
- Google Cloud Vertex AI SDK
- LangChain 0.1.0+
- PostgreSQL 14+
- Docker & Kubernetes
- Terraform (IaC)
